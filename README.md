# WedoMall
KoomShop est une marketplace multi-vendeurs permettant aux commerçants de créer leur boutique et de vendre leurs produits en ligne. Les utilisateurs peuvent parcourir les vendeurs, consulter les produits, ajouter au panier et finaliser leur commande. Une plateforme simple, rapide et inspirée des grands sites e-commerce africains comme Jumia.
mini-jumia/
│── index.html          (Accueil + vendeurs)
│── vendor.html         (Page boutique d’un vendeur)
│── product.html        (Page d’un produit)
│── cart.html           (Panier)
│── style.css
└── script.js
